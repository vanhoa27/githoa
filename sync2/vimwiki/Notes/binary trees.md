### Properties
- every object has a key
- comparable keys
- key -> data

### Operations
- O = Object
- Search(x): ouput key s(O)=x or nil(NULL)
- Insertion(x): insert O in data
- Delete(x): delete O from data 

### Defintion
- structure in finite set
- empty set = binary tree (leerer Baum)
