# Notetaking in Neovim

## Software used
- **TextEditor:** Neovim
- **Plugins:** Vimwiki, MarkdownPreview, LuaSnip
- **Mobile MarkdownEditor:** Markor
- **Syncing:** Syncthing

- insert Emojisyntax ([Syntax List](https://gist.github.com/rxaviers/7360908))
